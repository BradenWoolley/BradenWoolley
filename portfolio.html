Afronaughts

This is a multiplayer platformer built in Unity for Android and PC. I've never made a tutorial for a game before and decided that I would implement a simple tutorial using the environment for instructions.
In the main game I have incorporated a lobby and matchmaking system. I have replicated all player movement and am currently implementing the game loop logic.

Unreal Engine Multiplayer

I am currently building a multiplayer shooter in Unreal Engine 4 and C++. Currently I have implemented all basic movement mechanics and associated replication. I have also created a reusable health system and weapon system( which is reconfigurable to make multiple weapon types). I'm currently creating AI for players to fight against.

City Builder

I am currently working on a simple city builder in Unity. The game currently consists of different building types that can be expanded on by a designer. Additionally there is a simple economy that calculates income based on how many citizens and jobs are available and citizens can only be acquired if there is sufficient housing. Features to be added will be vehicles for resource gathering and trade.

Reflex

This was the first game that I published on the Google Play Store. The game incorporates third party services such as Unity Ads and Google Play Services for leaderboards. After launching I have updated the game to include a streak system and improved UI animations with a tweening library. I have had around +1500 downloads in 4 months with zero marketing.
